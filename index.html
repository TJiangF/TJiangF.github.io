Interactive Display with Lidar

It is a way to interact with any display remotely by waving or pointing with hand or other object. The lidar is used for hand detection and localization, and the data is sent back to host pc with a TTL converter, and the host PC is running a SFML Demo.

With all parameters set up, including display size, the distance between lidar and display edge, and the rotation for better leveling performance, it is a all-display compatible way to remotely interact.

Radio Delay Project

Built a device that delays the radio broadcast to synchronize with TV stream, it takes radio output signal as input, using DACs of ESP32-S3 chip to convert the signal to data, stores in the RAM and send out through PWM signal after the time being set, to headphones or speakers. The Bluetooth functions are supported as well.

Used DesignSpark to design the schematic and PCB. The input signal is amplified by opamps with digipots, keeping the input signal voltage in the range of ADC, then the digital data is stored in 8MB PSRAM of ESP32 chip. The transparent case is designed with Solidworks and printed by 3D printer.

The device is powered by a lithuim battery and charged by usb-c cable.

3D printer using concrete mateial

Built a 3d printed using concrete material as printing material, extruded by spinning screw, and the firmware was inspired by Marlin, modified for the adjustive offset in z axis, matching the physical characteristics of concrete material, since it collapses as it is being printed. The whole framework is 1.5*1.2*1.2m, and cross experiments were conducted to test their performance and feasibility.

A patent of the extruder which can be easily attached or replaced, with the adjustibilty of changing the shape and size of nozzle has been applied and approved.
